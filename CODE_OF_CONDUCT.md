# Code of Conduct

This project is intended to be a respectful, practical, and welcoming place to collaborate.

## Expected Behavior

- Be kind and professional.
- Assume good intent, especially when discussing security or reliability concerns.
- Keep feedback specific, actionable, and focused on the work.
- Respect different experience levels and operating environments.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Discriminatory language or behavior.
- Publishing private information without permission.
- Deliberately unsafe security advice or instructions.

## Reporting

If this project becomes public, replace this section with the preferred private reporting contact for Paravane Labs.
